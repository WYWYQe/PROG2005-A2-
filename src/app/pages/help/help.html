Help & Troubleshooting

This page provides common questions and troubleshooting guidance for using the inventory management system.

Frequently Asked Questions

Why can’t I add a new item?

Please check whether all required fields are completed. Item ID must be unique, and fields like item name, quantity, price, supplier, and stock status must be valid.

Why is my quantity or price rejected?

Quantity and price only accept valid numeric values. Negative numbers and text input are not allowed.

How do I update an item?

Go to the Inventory page, click the Edit button beside the item, make changes in the form, and then save the updated record.

How do I delete an item?

Click the Delete button beside the item you want to remove. A confirmation step is shown before the record is deleted.

How can I find popular items?

Use the Search page and choose the “Popular only” option to display items marked as popular.

Troubleshooting

Search returns no results

Make sure the item name is spelled correctly. Try using part of the name instead of the full name.

The page looks broken on a small screen

Refresh the browser and make sure the latest CSS changes are loaded. The system is designed to be responsive for mobile and desktop views.

My saved data disappeared

If the app only uses in-memory data, information will be lost after refresh or closing the browser. If localStorage is enabled, clearing browser data may also remove saved items.

How do I reset the application?

You can remove test items manually by deleting them one by one, or clear local storage if your version stores data in the browser.
